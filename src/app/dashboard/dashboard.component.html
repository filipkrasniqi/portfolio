Booksgram

Software developed together with two colleagues as a free-lancer. Software developed together with a colleague of mine as a free-lancer. We developed a website and an Android application providing a web radio service. Tracks are played according to pre-defined constraints on artists, genre etc. An administrator has the chance to add/remove the tracks, and to connect on the web radio to talk live.
Main contributions: backend implementation (NodeJS), website (HTML5, CSS, Angular).

Park&Light

Contributor when working for Emmequadro. This is part of the "Smart City" project. This application was able to control different centrals. A central could be a "Park" or a "Light". A "Park" central was able to capture whether some park slots were occupied and communicated this to the server. A "Light" central was able to control street lamps so that they could turn on/off (with a certain level of granularity on the settings).
Main contributions: backend (PHP), frontend (HTML5, CSS, JavaScript/JQuery).
