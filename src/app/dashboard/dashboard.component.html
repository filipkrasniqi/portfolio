High Capacity and Resilient Large-Scale Deterministic IP Networks

1 year research project at Huawei, Technologies (Paris) when working as an External Consultant - Software Engineer. We designed a novel network architecture to tackle the issue of deterministic networking without the use of Time Synchronized Networks (TSN); instead, we focused on the Deterministic IP network, proposing a novel approach that supports resiliency in large-scale deterministc networks. My contribution has been manyfold:
  1. Interaction with an ILP-based optimizer that takes in input traffic and network, and outputs the suboptimal solution for the Deterministic IP framework;
  2. Design and implementation of the architecture inside the Omnet++ simulator;
  3. Design and implementation of a framework that launches the optimizer, initializes the network and validates the simulation through exhausting simulations, with statistics that allow to motivate the implementation of the architecture;
  4. Contribution in writing the journal.
More details can be found here: https://link.springer.com/article/10.1007/s10922-022-09683-3.

Smart Campus: Parking & Directions

Design and implementation of a smart direction system with IoT technologies at PoliMI; design of a smart parking system. We successfully implemented a PoC that provides guidance to users inside the university; the user defines, through a smartphone, the final destination; from that moment on, he is tracked through beacon technology while a numerous amount of led matrices, spreaded all over the environment, show the next direction to the user. We considered different scenarios: multiple users, disconnection, need for system update etc. in order to simplify the initialization and setup of every single component. I designed the system with some colleagues in Politecnico di Milano, and I personally implemented the whole system. I exploited technologies such as Machine Learning, MQTT, Raspberry PIs; I collaborated with a group of designers and architects to properly integrate the system inside the university.
